Attach Ghost Key to specific fields

This example shows how to manually attach Ghost Key to specific password fields for more control.
// Initialize Ghost Key in manual mode
const ghostKey = new GhostKey({
  apiKey: 'your_api_key',
  mode: 'manual'
});

// Attach to specific field
ghostKey.attachToPasswordField('#password', {
  type: 'registration'
});
